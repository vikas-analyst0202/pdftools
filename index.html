User Guide

Privacy & Security

This tool works 100% Offline. Your files are processed entirely within your browser using WebAssembly. No documents are ever uploaded to any server. You can safely use this even in Flight Mode.

Merge Tool

Combine multiple PDFs into a single document.

  • Drag & Drop: Reorder files by dragging the thumbnails.
  • Rename: Double-click any filename to rename it before merging.
  • Hyperlinked TOC: Creates a clickable Table of Contents on the first page linking to each merged document.
  • Source Filename: Stamps the original filename on the top of each page for reference.

Split Tool

Extract specific pages or split a document into parts.

  • Single File: This tool processes one file at a time.
  • Page Range: Enter ranges like 1-5, 8, 11-13.
    • Leaving it empty splits every page into a separate file (zipped).
    • Entering 1-5 creates a single PDF with just those 5 pages.
    • Entering 1-5, 8 creates two PDFs (unless the result is just one file).

Compress Tool

Reduce file size by removing unnecessary data.

  • Strip Metadata: Removes properties like Author, Title, Creator.
  • Remove Annotations: Deletes comments, highlights, and sticky notes.
  • Flatten Forms: Converts editable form fields into static text (non-editable).
  • Remove Attachments: Strips any files embedded inside the PDF.

Installation

This is a Progressive Web App (PWA). You can install it on your device for offline access by clicking the Install App button in the top right (if supported by your browser).
